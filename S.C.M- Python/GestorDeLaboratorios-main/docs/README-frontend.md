README del frontend

Instrucciones rápidas:
- Ir a la carpeta frontend: `cd frontend`
- Instalar dependencias: `npm install`
- Levantar en modo desarrollo: `npm run dev`
- Build producción: `npm run build` (genera `dist/`)

Notas:
- Asegúrate de que el backend esté corriendo en http://127.0.0.1:5000 para que las llamadas a `/api/` funcionen en desarrollo.
